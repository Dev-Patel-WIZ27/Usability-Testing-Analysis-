# Usability Testing & Analysis

This project repository contains the complete documentation, planning, and results for the usability testing phase of a digital product. It is designed to fulfill the requirements of conducting user testing and analyzing the results for iterative design improvement.

## 📂 Project Structure

- **`usability_testing_report.md`**: The comprehensive final report containing findings, usability metrics, and design improvement suggestions.
- **`test_plan_and_scenarios.md`**: The structured plan detailing the testing methodology, user demographics, and specific tasks given to users.
- **`raw_feedback.md`**: Simulated raw feedback data collected during the testing sessions.

## 🎯 Objectives
- Prepare usability test plans and scenarios.
- Recruit sample users and conduct sessions.
- Record observations and collect feedback.
- Provide a report with findings and design improvement suggestions.

## 📊 Expected Outcomes
- Understand usability metrics.
- Execute proper testing methods.
- Propose iterative design improvements based on real data.
